
**Pre-cursor**
The valid_keywords.txt file is used to stop valid keywords from being incorrectly counted as errors. These are mostly brand terms. 
The output is BOOL format ie. 1 = error, 0 = no error

**Actions**
Paste your keywords in to the keywords.txt file 
Add keywords to the valid_keywords.txt file if necessary. 
Run the script
Copy & paste the output in to your keyword sheet
